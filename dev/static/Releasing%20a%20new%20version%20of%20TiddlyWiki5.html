Releasing a new version of TiddlyWiki5

9 days ago
  1. Adjust the release date of the latest release tiddler (eg, Release 5.0.7-beta)
  2. Ensure Releases has the new version as the default tab
  3. Adjust the modified time of HelloThere
  4. Make sure Jermolene/TiddlyWiki5 is fully committed
  5. Edit package.json to the new version number
  6. Run bin/bld.sh to build the deployment files
  7. Restore package.json to the previous version number
  8. Run bin/verbump "5.0.8-beta", substituting the new version number
  9. Run bin/deploy.sh
  10. Run bin/wbld.sh <username> <password>
