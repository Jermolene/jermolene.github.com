Improving TiddlyWiki Documentation

6 hours ago

Anyone can submit improvements to the TiddlyWiki documentation that appears on http://tiddlywiki.com.

  1. Create an account on https://github.com if you don't already have one
  2. On http://tiddlywiki.com, click "edit" on the tiddler you want to improve
  3. You should see a pink banner with the text: Can you help us improve this documentation? Find out how to edit this tiddler on GitHub
  4. Click on the external link ...this tiddler on GitHub
  5. A new browser tab should open ready to edit the tiddler on github.com
  6. Below the edit box for the tiddler text you should see a box labelled Propose file change
  7. Enter a brief title to explain the change (eg, "Clarify attribute syntax instability")
  8. If necessary, enter a longer description too
  9. Click the green button labelled Propose file change
  10. On the following screen, click the green button labelled Create pull request

Jermolene or one of the other core developers will then have the opportunity to merge your pull request so that it is incorporated into the next build of http://tiddlywiki.com.
